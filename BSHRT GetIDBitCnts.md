## API Call
BSHRT GetIDBitCnts(tsIDBitCnts *psIDBitCnts)
## DESCRIPTION
Get information regarding the ID bit and counts.

## PARAMETERS
psIDBitCnts pointer to structure to receive Bit Count information.

## RETURNS
TRUE = Success / FALSE = Failure
