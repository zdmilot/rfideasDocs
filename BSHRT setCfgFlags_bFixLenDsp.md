## API Call

## DESCRIPTION
C# interface for safe code. Set CfgFlags member&#39;s bFixLenDsp value. You must call ReadCfg() first. This function calls SetFlags() internally. Note: Send as fixed length with leading zeros as needed.

## PARAMETERS

## RETURNS
