## API Call
BSHRT pcSwipeSetFieldShowSepSen(WORD field, BSHRT enable)
## DESCRIPTION
Set the field Show Separators and Sentinels state.

## PARAMETERS
user field number 1..11, TRUE | FALSE

## RETURNS
TRUE = Success / FALSE = Failure
