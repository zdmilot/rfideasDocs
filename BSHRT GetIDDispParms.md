## API Call
BSHRT GetIDDispParms(tsIDDispParms *psIDDispParms)
## DESCRIPTION
Get display parameters.

## PARAMETERS
psIDDispParms pointer to structure to receive ID Display information.

## RETURNS
Returns non-zero on success, zero otherwise.
