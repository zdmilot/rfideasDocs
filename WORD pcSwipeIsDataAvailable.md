# WORD pcSwipeIsDataAvailable [Traffic]

## API Call
WORD pcSwipeIsDataAvailable(void)
## DESCRIPTION
Check if data is available on any track. If non zero then GetActiveID

## PARAMETERS
None

## RETURNS
0 = No Card Data Available.
