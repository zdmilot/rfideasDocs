## API Call
BSHRT GetUnsupportedProductErrorCode(void)
## DESCRIPTION
Read back the unsupported error code set by SetUnsupportedProductErrorCode().

## PARAMETERS
None

## RETURNS
BSHRT value set by SetUnsupportedProductErrorCode().
