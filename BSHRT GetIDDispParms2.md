## API Call
BSHRT GetIDDispParms2(tsIDDispParms2 *psIDDispParms)
## DESCRIPTION
Get display parameters 2.

## PARAMETERS
psIDDispParms2 pointer to structure to receive ID Display2 information.

## RETURNS
