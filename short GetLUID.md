# short GetLUID [Traffic]

## API Call
short GetLUID(void)
## DESCRIPTION
Get Logical Unit ID from last ReadCfg() of device.

## PARAMETERS
No parameters available.

## RETURNS
LUID a user defined 16-bit ID (0-65536) to be associated with the current selected device.
